# Screenshots

Coloque aqui as capturas de tela usadas no README principal. Use **estes nomes** (o README já os referencia):

| Arquivo | O que capturar |
|---------|----------------|
| `home.png` | Página inicial (hero + destaques) |
| `catalogo.png` | Listagem de produtos com filtros |
| `produto.png` | Página de detalhe do produto (variantes/tamanhos) |
| `checkout.png` | Carrinho ou checkout |
| `admin.png` | Painel administrativo (dashboard ou catálogo) |
| `tema.png` | Comparação tema claro vs. escuro (ou uma tela no tema escuro) |

**Dicas:**
- Largura ~1280px, formato PNG. Evite incluir dados pessoais reais nas capturas.
- Para uma imagem lado-a-lado (claro/escuro), pode montar uma colagem simples.
- Depois de adicionar, confira o README renderizado no GitHub.
